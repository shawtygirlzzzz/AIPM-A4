# Project Communication Plan

## Communication Matrix

| Communication Item | Audience | Frequency | Method |
| :--- | :--- | :--- | :--- |
| **Daily Stand-up / Task Sync** | Development Team, Project Manager | Daily (Weekdays) | MS Teams Chat / WhatsApp |
| **Code Commits & Pull Requests** | Development Team | As needed (Continuous) | GitHub (Issues & PRs) |
| **Lab Deliverables & Progress Reports** | Course Coordinator | Weekly | ULearn Submission |
| **WBS & Schedule Review** | Project Manager, Development Team | Weekly (Start of week) | MS Teams Meeting (Video) |
| **User Testing Feedback** | End Users | Once (During Validation Phase) | Google Forms / Direct Interview |
| **Risk & Blocker Alerts** | Project Manager | As needed (Immediate) | MS Teams (Direct Message) |
